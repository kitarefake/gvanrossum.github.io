Declaration Syntax

  • Two forms: inline and explicit

    • explicit form is easy to remove

  • Inline:

    • def gcd(a: int, b: int) -> int: ...

  • Explicit (two variants):

    • decl gcd: def(int, int) -> int def gcd(a, b): ...
    • def gcd(a, b): decl a: int, b: int decl return: int ...
