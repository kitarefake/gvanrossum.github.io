Typecase Strawman

    decl x: int | string | None | [any] | ....

    typecase x: int, i: print “an int: %d” % i string, s: print “a string: %s” % `s` None: print “nothing” else: print “don’t know”

  • All possible types must be accounted for, otherwise it’s a static error
